# Vídeos de prévia (VSL)

Coloque aqui vídeos curtos (~10s) para a seção **"Veja como tudo aconteceu"** de cada aventura.

- **Formato:** MP4 (recomendado), WebM ou OGG
- **Duração sugerida:** ~10 segundos (apenas para ilustrar a prévia)
- **Exemplo:** `preview-1.mp4`, `preview-2.mp4`

No admin, em **Vídeo de prévia / VSL (URL)**, use o caminho local, por exemplo:
- `/videos/preview-1.mp4`

O player da página usa o **player local** (HTML5) para esses arquivos.
